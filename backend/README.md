# Nutrition Elite Backend

## Requirements
- Node.js >= 18
- MongoDB

## Setup
1. cp .env.example .env
2. npm install
3. npm run dev

## Routes
- POST /api/auth/register
- POST /api/auth/login
- GET /api/plans
- POST /api/plans
- GET /api/plans/:id
- POST /api/plans/:id/unlock
- DELETE /api/plans/:id
